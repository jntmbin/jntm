# 🎨 猜画游戏 - Drawing Guessing Game

一个有趣的猜画游戏，你来画，AI来猜！

## 功能特点

- 🖌️ **自由绘画**：在画布上自由创作
- 🎨 **多种颜色**：8种颜色可选（黑、红、绿、蓝、黄、紫、青、橙）
- ✏️ **画笔工具**：可调节画笔粗细（1-20px）
- 🧹 **橡皮擦**：擦除不满意的部分
- 🗑️ **清空画布**：一键清空重新开始
- 🤖 **AI猜测**：画完后AI会分析你的画作并给出猜测结果
- 📱 **触摸支持**：支持触摸屏设备

## 如何使用

1. 在浏览器中打开 `drawing-game.html` 文件
2. 使用工具栏选择颜色和画笔粗细
3. 在画布上自由绘画
4. 画完后点击"让AI猜猜看"按钮
5. AI会分析你的画作并给出猜测结果

## AI猜测原理

AI通过分析以下特征来猜测画作内容：

- **颜色分析**：识别主要使用的颜色
- **覆盖率**：计算绘制内容占画布的比例
- **复杂度**：分析使用的颜色种类和绘制密度
- **启发式规则**：基于常见图案的特征进行匹配

## 技术栈

- 纯HTML5 + CSS3 + JavaScript
- Canvas API用于绘图功能
- 无需后端服务器，可直接在浏览器中运行

## 浏览器兼容性

支持所有现代浏览器：
- Chrome/Edge
- Firefox
- Safari
- 移动端浏览器
