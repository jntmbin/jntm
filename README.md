# 🎨 猜画游戏 - Drawing Guessing Game

一个有趣的猜画游戏，你来画，**真正的AI**来猜！使用 TensorFlow.js 和 MobileNet 深度学习模型进行图像识别。

## 功能特点

- 🖌️ **自由绘画**：在画布上自由创作
- 🎨 **多种颜色**：8种颜色可选（黑、红、绿、蓝、黄、紫、青、橙）
- ✏️ **画笔工具**：可调节画笔粗细（1-20px）
- 🧹 **橡皮擦**：擦除不满意的部分
- 🗑️ **清空画布**：一键清空重新开始
- 🤖 **真实AI识别**：使用 TensorFlow.js MobileNet 深度学习模型进行图像分类
- 🧠 **神经网络**：基于百万级图像训练的卷积神经网络
- 🌏 **中英双语**：AI识别结果自动翻译为中文
- 📱 **触摸支持**：支持触摸屏设备

## 如何使用

1. 在浏览器中打开 `drawing-game.html` 文件
2. 等待 AI 模型加载完成（首次加载需要几秒钟）
3. 看到 "✅ AI模型加载成功" 提示后即可开始
4. 使用工具栏选择颜色和画笔粗细
5. 在画布上自由绘画
6. 画完后点击"让AI猜猜看"按钮
7. AI 神经网络会分析你的画作并给出猜测结果

## AI 识别原理

本游戏使用 **真正的深度学习模型** 进行图像识别：

### MobileNet 神经网络
- **架构**：轻量级卷积神经网络（CNN）
- **训练数据**：ImageNet 数据集（超过 100 万张图像，1000 个类别）
- **模型大小**：约 16MB（优化后适合浏览器）
- **识别能力**：可识别动物、物体、食物、交通工具、家居用品等上千种物品

### 工作流程
1. 将画布图像缩放至 224×224 像素
2. 通过卷积神经网络提取图像特征
3. 使用全连接层进行分类
4. 输出前5个最可能的结果及其置信度
5. 自动将英文结果翻译为中文

### 特点
- ✅ **真实AI**：使用经过训练的神经网络，而非规则匹配
- ✅ **本地运行**：所有计算在浏览器中完成，无需后端服务器
- ✅ **实时识别**：通常在1-2秒内完成分析
- ✅ **高准确率**：对清晰的物体绘画有较高的识别准确率

## 技术栈

- **前端框架**：纯 HTML5 + CSS3 + JavaScript（无依赖）
- **AI 框架**：TensorFlow.js 4.15.0
- **AI 模型**：MobileNet v2.1.0（预训练图像分类模型）
- **绘图 API**：Canvas API
- **部署方式**：静态HTML文件，无需后端服务器
- **运行环境**：现代浏览器（支持WebGL加速）

## 浏览器兼容性

支持所有现代浏览器：
- Chrome/Edge（推荐）
- Firefox
- Safari
- 移动端浏览器

### 系统要求
- 需要 **稳定的网络连接**（首次加载模型需要下载约16MB）
- 支持 **WebGL** 的浏览器（用于GPU加速）
- 建议使用桌面浏览器以获得最佳体验

## 使用技巧

1. **画得清晰一些**：AI 对清晰、完整的物体识别效果更好
2. **尝试常见物品**：如猫、狗、汽车、苹果等，识别准确率更高
3. **居中绘制**：将主要对象画在画布中央
4. **适当填充**：不要画得太小，适当填充画布
5. **耐心等待**：首次加载模型需要几秒钟，请耐心等待

## 常见问题

**Q: 为什么 AI 识别不准确？**
A: MobileNet 主要是为照片设计的，对简笔画的识别能力有限。建议画得更清晰、更接近真实物体的样子。

**Q: 模型加载失败怎么办？**
A: 检查网络连接，刷新页面重试。确保浏览器支持 WebGL。

**Q: 可以离线使用吗？**
A: 首次使用需要联网下载模型（约16MB），浏览器会自动缓存模型，后续可能可以离线使用（取决于浏览器缓存策略）。
